# Love_Note_Generator

Description:
A love note generator made for my boyfriend, it includes a button to generate love notes and another to save those notes to a favorites section forever. 

Overview:
This was a personal project that showcases my ability to implement basic HTML, CSS, and JavaScript into a web application.

Features:
- Clean layout designed to reduce cognitive load
- Button to regenerate a variety of different love notes
- Button to save love notes to the favorites section
- A favorites section that keeps previously saved love notes indefinitely
- Floating heart animations present in the background when the user clicks the button to generate a new love letter

Tech Stack:
- HTML
- CSS
- JavaScript

Usage: 
1. Download or clone the repository
2. Navigate to the project folder
3. Open the HTML file in your web browser

> Note: This project is a static HTML file and does not require a web server or any additional setup.
