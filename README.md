# Love Note Generator
*Written by hand, sealed in wax* 

## Live Demo
> Coming soon, actively working on deployment: expect to see in update within 48 hours

## Concept
Love Note Generator began as a small gift for my boyfriend. What started as a bit of HTML, CSS, and JavaScript is now updated to feel more expansive and interactive than just a bubblegum pink background with simple clickable buttons. 
Inspired By:
- My boyfriend
- Handwritten love letters
- Keepsakes

## How It Works
Press the red wax seal, from there a newly generated love note appears, written as if it was typed on a typewriter. Press the postage stamp to keep the note of your choice indefinitely saved in your "Shoebox."

## Controls
* **Wax Seal** - Generates a new love note
* **Postage Stamp** - Saves your love note to your "Shoebox"

## Tech Stack:
- HTML
- CSS
- JavaScript

## Features:
- Typewriter content generation with a varied pace to micmick human typing
- Wax seal button with a press animation upon generating a new love note
- Postage stamp button with an inkblot bloom animation upon saving a love note
- "The Shoebox"; The section where all of the saved love notes go - your keepsakes
- SVG postmark stamped in the top right corner
- UI intended for emotional depth and a more tactile experience
- Love notes are kept across vists via "localStorage"

## Challenges + Learnings
- Translating physical objects into pure CSS without relying on images
- Using "clip-path" and radial gradients to suggest irregular, organic edges
- Designing typewriter motion that feels human rather than mechanical
- Choosing a typographic palette (Cormorant Garamond, EB Garamond, Special Elite) where each font carries a different emotional register
- Rewriting the entire aesthetic from scratch after outgrowing the original version, and learning that the interaction itself can be the emotional content
