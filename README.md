# Love_Note_Generator

Description:
A love note generator made for my lovely boyfriend, it includes a button to generate love notes and another to save those notes to a favorites section forever. 

Usage: 
1. Download or clone the repository
2. Navigate to the project folder
3. Open the HTML file in your web browser

> Note: This project is a static HTML file and does not require a web server or any additional setup.
